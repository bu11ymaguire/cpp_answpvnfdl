개강전 본격적인 동아리 활동에 앞서서 여름방학에 동아리에서 주관한 숭실대-고려대-한양대 연합 대회인 숭고한 알고리즘 대회에 참여하였다. 
내 백준 티어가 골드1이기 때문에 골드 티어까지 참여할 수 있는 디비전 3에 참여하였다.
최종 순위는 16팀(22팀 중 6팀이 탈주) 중에서 11팀으로 8문제 중에서 4문제를 맞췄고 동점자 5명 중에서 시간 관리에서 밀려서 11등을 하게 되었다.
평소에 백준 문제를 풀다가 이런식으로 대회에 참여해본 것은 처음이었는데, 중도나 스카에서 노래들으면서 끄적이는거랑 체감 난이도가 크게 달랐다.
