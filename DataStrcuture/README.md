편입 후 수강하였던 자료구조(CSE2010) 과목과 데이터사이언스 학부 알고리즘 스터디(데코더)에서 학습한 자료구조에 대해서 정리한 경로이다.

개발자의 가장 중요한 펀더멘털 중 하나가 자료구조를 비롯한 기초 CS 지식이라고 해서, 첫 학기 동안 스터디에 들어가 백준 문제도 풀어보고 자료구조 강의를 들었다.

연결리스트, 큐, 스택, 그래프의 구현 등을 다룬 중간고사에서는 평균 이상의 좋은 성적을 받았다. 그러나 다익스트라, 해시테이블, 허프만 코드 등 꼼꼼하게 공부하지 않으면 실수하기 쉬운 자료구조들을 제대로 공부하지 않아 기말고사에서 실수가 많았던 것 같다.

이러한 실패를 극복하기 위해 2학기부터는 데코더 스터디장이 임원으로 있는 알고리즘 동아리에 가입하여 활동하였다.
